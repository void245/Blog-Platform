## 🎨 Customization Tips

To customize colors, edit the CSS variables at the top of `enhancements.css`:

```css
:root {
  --primary-gradient: linear-gradient(135deg, #0085A1 0%, #006585 100%);
  --secondary-gradient: linear-gradient(135deg, #f5f7fa 0%, #f0f2f5 100%);
  --accent-gradient: linear-gradient(135deg, #00d4ff 0%, #0085A1 100%);
}
```
